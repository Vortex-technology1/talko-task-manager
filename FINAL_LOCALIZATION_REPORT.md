## ТЕХНІЧНІ ДЕТАЛІ

### Як працює система перекладів:

```javascript
// 1. window.t(key) шукає переклад
window.t = function(key) {
    const lang = window.currentLanguage || 'ua';
    const translations = window.translations || {};
    const langPack = translations[lang] || {};
    // Fallback на UA якщо ключ відсутній
    return langPack[key] || translations['ua']?.[key] || key;
};

// 2. При зміні мови оновлюються всі елементи з data-i18n
document.querySelectorAll('[data-i18n]').forEach(el => {
    const key = el.getAttribute('data-i18n');
    el.textContent = window.t(key);
});
```
