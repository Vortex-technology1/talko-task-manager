### 3. ⚠️ Захардкоджені українські тексти в HTML

**Результат:** ЗНАЙДЕНО 38 МІСЦЬ (некритично)

**Розподіл:**
- index.html: 27 проблем
- biz-structure.html: 11 проблем

**Типи проблем:**
1. **aria-label** (22 проблеми) - accessibility атрибути для screen readers
2. **title** (11 проблем) - tooltip при наведенні
3. **placeholder** (5 проблем) - підказки в полях вводу

**Приклади:**
```html
<!-- index.html -->
<input placeholder="Назва компанії">
<button aria-label="Закрити діалог">
<button title="AI Підтримка 24/7">
<button aria-label="Мій день">

<!-- biz-structure.html -->
<button title="Додати нову функцію" data-title="toolbar.add_function">
<button aria-label="Додати функцію">
<input placeholder="Наприклад: Відділ продажів">
<div aria-label="Обрати колір">
```

**Аналіз критичності:**
- ❌ **НЕ критично** для більшості користувачів
- ⚠️ **Впливає** на користувачів screen readers (accessibility)
- ⚠️ **Впливає** на tooltips та placeholder тексти
- ✅ **НЕ впливає** на основний контент інтерфейсу (він перекладається через data-i18n)

### 4. ✅ Fallback тексти в JS

**Результат:** FALLBACK НЕ СПРАЦЬОВУЄ (всі ключі є)

Знайдено **40+ fallback** конструкцій типу:
```javascript
window.t('key') || 'Українська резервна'
```

Але оскільки **всі RU ключі присутні** в translations.ru, fallback на українську **НЕ спрацьовує**.

**Приклади:**
```javascript
window.t('finLoading') || 'Завантаження...'  // ✓ ключ є в RU
t('saveWord') || 'Зберегти'                   // ✓ ключ є в RU
window.t('noUsers') || 'Немає користувачів'  // ✓ ключ є в RU
```

### Варіант А: Залишити як є (РЕКОМЕНДОВАНО)

**Обґрунтування:**
- Основний контент перекладається правильно

### Варіант Б: Виправити (якщо потрібна 100% локалізація)

**Що потрібно:**
1. Додати data-i18n-aria-label атрибути (22 місця)
2. Додати data-i18n-placeholder атрибути (5 місць)
3. Додати data-i18n-title атрибути (11 місць де їх ще немає)
4. Додати відповідні RU переклади в translations.ru
5. Розширити applyTranslations() функцію для обробки цих атрибутів

**Приклад виправлення:**
```html
<!-- До: -->
<button aria-label="Закрити діалог">&times;</button>

<!-- Після: -->
<button data-i18n-aria-label="closeDialog" aria-label="Закрити діалог">&times;</button>
```

```javascript
// В translations:
closeDialog: 'Закрити діалог',  // UA
closeDialog: 'Закрыть диалог',  // RU
```
